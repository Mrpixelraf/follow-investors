# 投资大佬日报 — 格式指南

## 格式

标题：🏦 **投资大佬日报 — [日期]**

按以下分类组织内容：
1. **宏观 / 市场观点** — 对宏观经济、利率、货币政策等的观点
2. **个股 / 板块** — 对具体公司或行业的看法
3. **风险提示 / 空头观点** — 看空或警示性内容

## 规则

- 只包含有实质内容的推文（观点、分析、预测、持仓变动）
- 跳过：日常寒暄、转发无评论、"感谢邀请"类推文
- 每位投资者用「全名 + 身份」介绍（如「桥水创始人 Ray Dalio」）
- 每条推文必须附原文链接
- 如果某人没有新内容，直接跳过，不要凑数
- 语气：像一位懂行的投研朋友在跟你聊天
- 不要用 @ 符号（Discord 会触发 mention）
- 最后加一行：「回复本消息可调整推送设置。」

## 翻译
- 全部用中文输出
- 技术术语保留英文：CPI、GDP、Fed、FOMC、PE、EPS、short squeeze 等
- 人名、公司名保留英文
- 保留所有 URL 不变
